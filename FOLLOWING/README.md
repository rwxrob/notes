# Following

Public knowledge repos followed as git submodules. The directory
structure mirrors the full URL path (e.g.
`FOLLOWING/github.com/rwxrob/notes-health`), supporting repos from any
git host.

To fetch all submodules after cloning this repo:

```sh
git submodule update --init --recursive
```

To update all submodules to their latest commits:

```sh
git submodule update --remote --merge
```
