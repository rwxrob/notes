# Notes on Tech Topics

Public tech knowledge base — notes, references, and ideas on technology topics. Part of a broader PKM system using Git + Obsidian + GitHub Copilot.

## Structure

- `AGENTS.md` — instructions and conventions for AI agents working in this repo
- `DISCOVER.md` — prompts to discover new public knowledge repos to follow
- `FOLLOWING/` — git submodules of followed public knowledge repos

## Cloning

To clone with all followed repos:

```sh
git clone --recurse-submodules https://github.com/rwxrob/notes-tech
```

Or after a plain clone:

```sh
git submodule update --init --recursive
```
